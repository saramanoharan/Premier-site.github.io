Quel est le travail d'un gestionnaire de bases de données ?

Le gestionnaire base de données est responsable du système de gestion, s’assure de la sécurité des données numériques de son entreprise et rend ces informations accessibles aux collaborateurs. Le gestionnaire base de données est également connu sous le nom d’administrateur de base de données.

Comment devenir un gestionnaire de bases de données ?

Un gestionnaire base de données a généralement suivi un parcours universitaire et obtenu un master (bac +5) en informatique avec une spécialisation en Systèmes d'Information et Analyse Décisionnelle. Cependant d’autres parcours sont envisageables à condition d’avoir suivi une spécialisation en exploitation de bases de données et une expérience entre 2 et 3 ans dans ce milieu.
